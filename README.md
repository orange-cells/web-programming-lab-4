**_Описание_**:

Репозиторий содержит код приложения для вывода прогноза погоды на 3 дня.

**_Функционал_**:

1. Запрос доступа к геопозиции пользователя при открытии приложения. При отказе доступа возможность ввести свое местоположение вручную.
2. Возможность добавлять карточки с прогнозом для других городов.
3. Обновление данных при нажатии на кнопку (через повторный запрос).
4. Сохранение информации, введенной пользователем, при обновлении страницы.

**_Технологический стек_**:

Работа выполнена с использованием HTML, CSS, JavaScript.

**_Использование API:_**:

Данные о прогнозе погоды, а также координаты города, название которого введено пользователем, получаются при помощи API https://open-meteo.com. Название города по координатам геолокации пользователя определяется через API openstreetmap.org. 

**_Адаптивный дизайн_**:

Обеспечено корректное отображение сайта на разных устройствах (ноутбук, планшет и смартфон).
